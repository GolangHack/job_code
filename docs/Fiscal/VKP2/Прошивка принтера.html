Обновление прошивки принтера Custom VKP80II.

Для корректной работы данного принтера с ККТ, необходимо обязательно обновить прошивку, независимо от вашей версии, в данной прошивке исправлены проблемы с отображением отдельных символов

1.Вся ответственность за проведение операции обновления встроенного программного обеспечения принтера (прошивки) лежит на исполнителе операции.
Рассматривается случай, когда принтер подключается к компьютеру посредством USB-интерфейсного кабеля, принтер исправен, бумага загружена, светодиод STATUS горит постоянно зеленым светом.

2.Рекомендуется проводить обновление прошивки принтера на отдельном компьютере (ноутбуке), так как обновление прошивки требует установленного драйвера, а в некоторых случаях работа прикладного ПО терминала исключает использование драйвера принтера. Более того, в некоторых случаях применяются системы контроли целостности ПО, которые блокируют установку/удаление критических компонентов ПО.
2.1.Чтобы узнать текущую версию прошивки: выключить принтер → нажать и удерживать кнопку LF → подключить к принтеру питание → после начала печати отпустить кнопку LF→ принтер напечатает страницу SETUP → нажать кнопку LF для отрезания бумаги. В верхней части чека в рамочке будет информация о версии прошивки, например VKP80 II - rel 4.20, где 4.20 - версия прошивки.
2.2.Если у Вас прошивка версии 2.28 и 2.37 и 4.02 или после модели принтера присутствуют символы STY - у Вас залоченный принтер. Необходимо связаться с нами.

3.Установите драйвер принтера 254-diw_vkp80ii_301.

4.Перед началом установки драйвера отключить интерфейсный USB-кабель принтера от компьютера, на котором будет производится установка драйвера. В процессе инсталляции по запросу установщика драйвера подключить USB-кабель принтера к компьютеру.

5.После окончания установки драйвера принтера перезагрузить компьютер, зайти в список доступных принтеров, убедится, что принтер CUSTOM VKP80 II присутствует в списке, выполнить печать тестовой страницы: правой кнопкой мыши нажать на иконку принтера → Свойства принтера → вкладка Общие → Пробная
печать. Если тестовая страница успешно была распечатана, переходите к следующему пункту.

6.Скачайте программу обновления прошивки принтера UpgCePrninstall_249.После ее установки проделайте следующее:
6.1.Пуск→ Все программы →Custom→UpgCePrn→UpgCePrn
6.2.В открывшемся окне нажать кнопку «Select *.psw» и выбрать подходящую прошивку VKP80_2_420_20161107.
6.3.Нажать кнопку BeginUpgrade→ выбрать USB-порт → Нажать ОК → в окне выбора принтера убедится, что выбран CUSTOM VKP80 II → Нажать ОК → Еще раз ОК → В грозном окне «Warning!: Theupgradewillbenexttobegin…» снова нажать ОК → Снова ОК
6.4.Вся процедура обновления прошивки может занимать до 5 минут, при это индикатор Status на самом принтере должен быстро моргать зелено-красным огнем.
6.5 После завершения процедуры появится окно с сообщением Upgradesuccessfullycompleted. Можно нажать ОК и закрыть программу обновления.

7. Убедится, что статусный светодиод принтера постоянно горит зеленым светом и отключить интерфейсный USB-кабели и кабель питания.
8. Подключить интерфейсный USB-кабель и кабель питания, после чего, согласно описанию, в п. 2.1 убедится, что версия прошивки изменилась.

P.S. Для удаления всех, раннее установленных драйверов для термопринтеров Custom, используйте утилиту CEDrvRT_2-00.

Прошивка принтера Citizen CT-S2000

1.Вся ответственность за проведение операции обновления встроенного программного обеспечения принтера (прошивки) лежит на исполнителе операции.
Рассматривается случай, когда принтер подключается к компьютеру посредством USB-интерфейсного кабеля, принтер исправен, бумага загружена, светодиод POWER горит постоянно зеленым светом.

2.Рекомендуется проводить обновление прошивки принтера на отдельном компьютере (ноутбуке), на котором ни разу не устанавливались драйвера чекового принтера с установленной ОС Windows XP. Если принтер ранее уже был установлен в системе, для корректной работы утилиты драйвер принтера необходимо удалить. Также необходимо удалить принтер на вкладе "Устройства и принтера" в "Панели управления".
2.1.Чтобы узнать текущую версию прошивки: выключить принтер → нажать и удерживать кнопку FEED → подключить к принтеру питание → после звукового сигнала отпустить кнопку FEED → принтер напечатает страницу с установленными параметрами. В верхней части чека будет информация о версии прошивки, например ROM VERSION: CS10-0900, где 10-0900 - версия прошивки.

3.Скачайте программу обновления прошивки принтера SetupCSJPrnCS-E. Для запуска файла установки в ОС Windows 7/8/10 необходимо зайти в свойства данного файла. Открыть вкладку "Совместимость" и поставить галочку "Запускать программу в режиме совместимости с Windows XP (пакет обновления 3)" и нажать ОК. Запуск программы необходимо производить от имени администратора. После её установки проделайте следующее:
3.1.Пуск → Все программы → Citizen → Printer Customization.
3.2.В открывшемся окне нажать кнопку I/F → Interface Selection → выбрать USB Interface. Далее Port No. → выбрать номер порта → Select → OK. (Для того чтобы узнать номер USB-порта необходимо открыть Диспетчер устройств → Контроллеры USB → Поддержка USB-принтера → вкладка Общие → Размещение: Port_#0002.)
3.3.Далее нажать кнопку Model → CT-S2000 STD → Select → OK.
3.4.Нажать кнопку "i" → появится информация о принтере, включая версию прошивки (F/W Version).
3.5.Далее File → Firmware Load → … → тип файла меняем на All Files → выбрать файл PC_CS10-0900_ALL.ARC → Start F/W Loading → OK.
3.6.Вся процедура обновления прошивки может занимать до 10 минут, при это в окне программы сначала отобразится Now Loading → затем Writing F/W индикатор POWER при этом будет моргать зеленым цветом.
3.7.После завершения процедуры прошивки кратковременно появляется надпись F/W Loading Completed, а принтер издаст механический звук.

4.Убедится, что светодиод принтера POWER постоянно горит зеленым светом после чего закрыть окно и выключить принтер.

5.Далее согласно описанию п.2.1 или п.3.4 убедится, что версия прошивки изменилась.

Обновление прошивки принтера SNBC BT-T080R

2.Рекомендуется проводить обновление прошивки принтера на отдельном компьютере (ноутбуке), так как обновление прошивки требует установленного драйвера, а в некоторых случаях работа прикладного ПО терминала исключает использование драйвера принтера. Более того, в некоторых случаях применяются системы контроли целостности ПО, которые блокируют установку/удаление критических компонентов ПО.
2.1.Чтобы узнать текущую версию прошивки: выключить принтер → нажать и удерживать кнопку FEED → подключить к принтеру питание → после начала печати отпустить кнопку FEED → принтер напечатает страницу ***BT-T080R TEST FORM***. В верхней части чека будет информация о версии прошивки, например Main Firmware: FV1.040 где FV1.040 - версия прошивки.
2.2. Если вместо данной страницу напечатано MAIN MENU, необходимо два раза кратковременно нажать кнопку FEED (соответствует меню Print Self Test), после чего еще раз нажать и удерживать кнопку FEED не менее 1 с, для подтверждения выбранного действия. Принтер напечатает страницу ***BT-T080R TEST FORM***.

3.Установите драйвер принтера Setup_BT-T080R V1.10.

4.Перед началом установки драйвера отключить интерфейсный USB-кабель принтера от компьютера, на котором будет производится установка драйвера. В процессе инсталляции по запросу установщика драйвера подключить USB-кабель принтера к компьютеру.

5.После окончания установки драйвера принтера перезагрузить компьютер, зайти в список доступных принтеров, убедится, что принтер BT-T080R(U) 1 присутствует в списке, выполнить печать тестовой страницы: правой кнопкой мыши нажать на иконку принтера → Свойства принтера → вкладка Общие → Пробная печать. Если тестовая страница успешно была распечатана, переходите к следующему пункту.

6.Скачайте программу обновления принтера KIOSKUtility V2.67.
6.1.Запустите файл KIOSKUtility.exe в папке KIOSKUtility V2.67. Далее нажимаем File → Open port → вверху выбираем модель принтера BT-T080R → выбрать USB → Ok.
6.2.Выбираем Printer Status → Printer Infor → Ок. Появится информация о ККТ, где Firmware Printer: FV1.040 - версия прошивки принтера.
6.3.Выбираем Setting → Download → Browse → выбрать файл BT-T080R_FV1040_KSKQV12_MAIN_T06 → Download.
6.4.После завершения процедуры появится окно с сообщением Download  Success! Please reopen the port. Нажать ОК → Exit → Exit.
6.5.Далее необходимо загрузить файл конфигурации. Выполнить п.6.1. Выбираем Setting → Download → Browse → меняем тип файла на All Files → выбрать файл Configuration setting by feed key button → Download.
6.6.После завершения процедуры появится окно с сообщением Download  Success! Нажать ОК → Exit → Exit.
